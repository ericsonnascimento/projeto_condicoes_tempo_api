# Projeto ADA em parceria com o Santander: Desenvolvimento de Site de Consulta de Condições Climáticas 

* Breve Resumo:

Projeto de desenvolvimento de website onde foram desenvolvidas diversas skills relacionas a HTML, CSS e TypeScript utilizando ferramentas de conexão assincrona a API openweathermap.org retornando temperatura, local da pesquisa e ícone diretamente na página web utilizando arquiteturas:

* async / await
* função fetch
* try / catch
